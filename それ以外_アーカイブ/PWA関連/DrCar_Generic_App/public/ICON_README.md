# アイコン画像について

## 必要なファイル
- `icon-512.png`: 512x512ピクセルのPNG画像

## 作成方法
1. 512x512ピクセルの正方形の画像を作成
2. ドクターカーまたは医療関連のアイコンを配置
3. PNG形式で保存
4. `public/icon-512.png`として配置

## 推奨デザイン
- 背景色: #f2f2f7（アプリのテーマカラーに合わせる）
- アイコン: 救急車や医療関連のシンプルなデザイン
- 透明背景も可
