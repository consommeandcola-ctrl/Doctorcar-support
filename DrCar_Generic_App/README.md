# 🚑 ドクターカー活動記録アプリ (汎用版) / Dr.Car Record App (Generic)

救急現場、ドクターカー、ドクターヘリ活動における記録・報告書作成を支援するPWA（Progressive Web App）です。
オフライン動作に対応し、タップ操作だけで迅速にバイタルサインや処置の記録、経過表の作成が可能です。

## ✨ 特徴

* **完全オフライン動作**: 電波の届かない現場でもすべての機能が利用可能です。
* **地域設定のカスタマイズ**: アプリ内の「設定」タブから、地域の医療機関名や救急隊名を自由に変更・保存できます（プログラミング不要）。
* **直感的なUI**: 揺れる車内でも操作しやすい、iOSスタイルの大型ボタンとダイヤル入力。
* **自動計算機能**: GCS/JCS、CPA時のNo Flow/Low Flow Time、NIHSSスコアなどを自動計算します。
* **詳記生成AIロジック**: タップ選択だけで、保険請求や搬送選定理由に適した「症状詳記」の文章をテンプレート生成します。

## 📲 インストール方法 (PWA)

本アプリはストアからダウンロードするほか、ブラウザから直接インストールして使用できます。

1.  Safari (iOS) または Chrome (Android/PC) でアプリのURLを開きます。
2.  **iOSの場合**: 「共有」アイコン → 「ホーム画面に追加」をタップします。
3.  **Androidの場合**: メニューから「アプリをインストール」または「ホーム画面に追加」をタップします。

## ⚙️ 初期設定 (地域データの登録)

初めて使用する際は、以下の手順で地域のマスタデータを設定してください。

1.  画面下部のメニューから**「設定 (歯車アイコン)」**タブを開きます。
2.  **「病院リスト」**: 搬送候補となる病院名を1行に1つずつ入力します。
3.  **「救急隊リスト」**: 連携する消防・救急隊名を1行に1つずつ入力します。
4.  **「保存」**ボタンを押し、アプリを再読み込み（リロード）すると反映されます。
